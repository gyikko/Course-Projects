 PauseCommand (VisualizationSimulator *sim)
 This class is mainly designed to connect front and back end. The functions in it provides buttons for users to click to achieve functionalities built in the code. By clicking the pause button, user can pause the simulator and he or she can also resume the simulator by clicking the resume button. So, in my_web_server_command, I created a new class PauseCommand, which takes VisualizationSimulator as a parameter to connect with front end.
